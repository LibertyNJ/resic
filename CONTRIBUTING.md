# Contributing

Thank you for your interest in contributing to RESIC.

## Code of Conduct

All contributors are expected to abide by the
[code of conduct](CODE_OF_CONDUCT.md).

## Security Vulnerabilities

If you believe that you have found a security vulnerability,
**do not open a bug report**. Please follow the instructions in the
[security policy](SECURITY.md) instead.

## Bug Reports

Please check if the
[bug has already been reported](https://github.com/LibertyNJ/resic/issues?q=is%3Aopen+is%3Aissue+label%3Abug)
first. If it has not,
[report a bug here](https://github.com/LibertyNJ/resic/issues/new?template=BUG_REPORT.yml).
