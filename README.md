# RESIC

A Rust client for
[EVE Stable Infrastructure (ESI)](https://developers.eveonline.com/api-explorer)
by [CCP Games](https://www.ccpgames.com/).

## About

RESIC seeks to offer a client with simple, intuitive implementations of
[ESI endpoints](https://developers.eveonline.com/api-explorer). The client is
designed to make it easy to follow
[ESI best practices](https://developers.eveonline.com/docs/services/esi/best-practices/).
